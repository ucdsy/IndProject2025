# 网上模板与公开范例汇总（2026-04-06）

> 用途：为当前发明专利申请文件起草提供格式参考。分为“官方模板/官方规范”和“公开专利范例”两类。前者用于对齐提交格式，后者仅用于学习体例，不用于照抄内容。

## 1. 官方模板与官方规范

### 1.1 国家知识产权局表格下载总入口
- 用途：
  - 下载《发明专利请求书》
  - 下载《权利要求书》
  - 下载《说明书》
  - 下载《说明书摘要》
  - 下载《说明书附图》
- 官方入口：
  - [CNIPA 表格下载栏目](https://www.cnipa.gov.cn/col/col192/index.html)

### 1.2 国家知识产权局关于请求类表格启用版本的通知
- 用途：
  - 确认纸件请求书使用的版本是否为当前有效版本
  - 避免使用旧版请求书
- 参考：
  - [CNIPA：关于修订七张专利申请请求类表格的通知（2025-01-02）](https://www.cnipa.gov.cn/art/2025/1/2/art_75_197001.html)

### 1.3 专利申请文件官方组成与顺序
- 用途：
  - 确认提交文件种类
  - 确认纸件文件排序
- 参考：
  - [CNIPA：专利申请相关事项介绍](https://www.cnipa.gov.cn/art/2020/6/5/art_1517_92472.html)

### 1.4 国家知识产权局办事指南
- 用途：
  - 一页看清发明专利相关表格名称
  - 用于核对请求书、权利要求书、说明书、说明书摘要、说明书附图的标准名称
- 参考：
  - [《知识产权政务服务事项办事指南（第二版）》PDF](https://www.cnipa.gov.cn/module/download/downfile.jsp?classid=0&filename=248ec440950744e3905ec2553c3461a7.pdf&showname=%E3%80%8A%E7%9F%A5%E8%AF%86%E4%BA%A7%E6%9D%83%E6%94%BF%E5%8A%A1%E6%9C%8D%E5%8A%A1%E4%BA%8B%E9%A1%B9%E5%8A%9E%E4%BA%8B%E6%8C%87%E5%8D%97%EF%BC%88%E7%AC%AC%E4%BA%8C%E7%89%88%EF%BC%89%E3%80%8B.pdf)

## 2. 官方写法范例

### 2.1 计算机程序相关发明的权利要求写法
- 用途：
  - 学“方法 + 装置/设备 + 存储介质 + 程序产品”这类组合写法
  - 学独立权利要求的正式长句体例
- 最有用的点：
  - 直接给出官方示例：
    - `一种……方法，其特征在于，包括以下步骤：……`
    - `一种计算机装置/设备/系统……`
    - `一种计算机可读存储介质……`
- 参考：
  - [CNIPA：《专利审查指南》（2023）修改解读（四）](https://www.cnipa.gov.cn/art/2024/1/18/art_2199_189877.html)
  - [CNIPA 公开材料：涉及计算机程序权利要求写法示例（PDF）](https://www.cnipa.gov.cn/module/download/downfile.jsp?classid=0&filename=419d2989347f4e699280e181ac987da3.pdf&showname=%E9%99%84%E4%BB%B61%EF%BC%9A%E3%80%8A%E4%B8%93%E5%88%A9%E5%AE%A1%E6%9F%A5%E6%8C%87%E5%8D%97%E4%BF%AE%E6%94%B9%E8%8D%89%E6%A1%88%EF%BC%88%E5%BE%81%E6%B1%82%E6%84%8F%E8%A7%81%E7%A8%BF%EF%BC%89%E3%80%8B%E4%BF%AE%E6%94%B9%E5%AF%B9%E7%85%A7%E8%A1%A8.pdf)

### 2.2 权利要求书体例规则
- 用途：
  - 学独立权利要求的前序部分和特征部分
  - 学从属权利要求的引用部分和限定部分
  - 学方法权利要求和产品权利要求的区别
- 参考：
  - [CNIPA 公开培训材料：权利要求的撰写和审查（PDF）](https://www.cnipa.gov.cn/transfer/pub/old/wxfw/zlwxxxggfw/gyjz/gyjzkj/201406/P020140624545868316293.pdf)
  - [CNIPA：专利申请知识问答](https://www.cnipa.gov.cn/art/2017/3/1/art_2651_167330.html)

### 2.3 说明书和附图写法规则
- 用途：
  - 学说明书应当如何围绕技术问题、技术方案、实施方式展开
  - 学流程图、结构框图、装置图在软件/控制类发明中的常见表达
- 参考：
  - [CNIPA：专利申请知识问答](https://www.cnipa.gov.cn/art/2017/3/1/art_2651_167330.html)
  - [CNIPA：专利申请相关事项介绍](https://www.cnipa.gov.cn/art/2020/6/5/art_1517_92472.html)

## 3. 公开专利范例

> 这些不是模板文件，而是已经公开的申请文本或公开摘要，可用于观察标题、摘要、技术领域、背景技术、权利要求保护主题的写法。

### 3.1 与软件/智能问答接近的范例
- [CN110321416A：基于AIML的智能问答方法、装置、计算机设备及存储介质](https://patents.google.com/patent/CN110321416A/zh)
- 可参考点：
  - 标题命名方式
  - 摘要写法
  - “方法、装置、设备及存储介质”组合保护主题

### 3.2 与智能体主题接近的范例
- [CN119025412A：智能体处理方法、装置、设备及存储介质](https://patents.google.com/patent/CN119025412A/zh)
- 可参考点：
  - 智能体相关发明如何命名
  - 方法类发明的权利要求和说明书章节体例

### 3.3 与算法/识别类方法接近的范例
- [CN116192439A：恶意网址的识别方法、装置、设备及存储介质](https://patents.google.com/patent/CN116192439A/zh)
- [CN111476035B：中文开放关系预测方法、装置、计算机设备和存储介质](https://patents.google.com/patent/CN111476035B/zh)
- 可参考点：
  - 算法型发明如何落到技术问题和技术效果
  - 如何避免标题和权利要求写成空泛“方案”

### 3.4 与“方法 + 系统 + 设备 + 介质”组合写法接近的范例
- [CN113992768A：一种TTE网络远程管理方法、系统、设备和可读存储介质](https://patents.google.com/patent/CN113992768A/zh)
- 可参考点：
  - 同一发明构思下，方法与系统权利要求如何配套

## 4. 模板怎么用

### 4.1 请求书
- 直接使用 CNIPA 官方表格，不自己造模板。

### 4.2 权利要求书
- 主模板：
  - 用官方示例和审查指南示例学句式。
- 参考顺序：
  1. 先看官方示例中的“方法 + 设备 + 介质”体例
  2. 再看公开专利中的章节长度和句式
  3. 最后回到自己的技术方案重写

### 4.3 说明书
- 不建议拿第三方“万能说明书模板”直接套。
- 最稳妥的做法是：
  - 按 CNIPA 通常章节顺序写：
    - 技术领域
    - 背景技术
    - 发明内容
    - 附图说明
    - 具体实施方式

### 4.4 摘要
- 看公开专利摘要，学“技术问题 + 核心步骤 + 技术效果”的压缩写法。
- 摘要不能写成宣传稿，也不能把所有从属特征都塞进去。

## 5. 对我们当前项目最有用的三类模板

### 第一优先级
- CNIPA 官方表格下载页
- 用来处理请求书和纸件格式

### 第二优先级
- CNIPA 审查指南的计算机程序权利要求示例
- 用来处理方法、系统、介质的权利要求结构

### 第三优先级
- 公开的智能体/软件类中国专利文本
- 用来学标题、摘要、背景技术和章节展开方式

## 6. 当前建议

- 请求书：直接用官方表格
- 权利要求书：以官方示例为主，公开专利范例为辅
- 说明书：由我们基于当前项目内容重写，不直接照搬所谓“万能模板”
- 公开专利范例只学格式，不学具体技术特征
